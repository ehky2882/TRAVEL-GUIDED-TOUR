# Rome — batch 1 (25 single-stop tours) — STAGED, awaiting narration MP3s 🇮🇹

The whole Rome single-stop set. All single-stop, geofenced (radius 30 m), free, under a new
maker **Atlas Studio ROM** 🇮🇹 (10th/next maker — created at wire-in). Heroes + galleries
sourced (Unsplash/Pexels/Pixabay = ship-safe; a handful Wikimedia CC — see CREDITS.md) or
owner-supplied, cropped 1200×900 WebP, pushed to gh-pages, verified live.
Image URL base: `https://ehky2882.github.io/TRAVEL-GUIDED-TOUR/images/`

Script + `_TTS.txt` pairs are in this folder (`NN_name.txt`). `transcriptText` = the verbatim
display `.txt` (outer whitespace trimmed, drop the `[beat]` line). Audio slug = the image slug.
